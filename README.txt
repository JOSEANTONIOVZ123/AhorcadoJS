Codigo creado solamente en JS con posible implementación de una API para las palabras

puedes ejecutar el código con node ejecutandolo con: node "nombreDelArchivo".js

Hay 3 archivos:

-> ahorcado.js el cual es el código el cual esta el ahorcado completo pero con un código burdo y tosco.
-> optimizado.js el cual es el mismo código con la misma funcionalidad pero más escueto 
-> mignify.js es una versión minificada del optimizado.js para que ocupe poco espacio. (3KB)
